# Game Server Status Plugin

游戏服务器状态监控插件 - 用于 Windrose 服务器

## 架构

```
┌─────────────────────┐         ┌─────────────────┐
│  Windows Server     │         │      Mac        │
│  39.107.87.36       │         │  定时任务 (5分钟) │
│                     │◀────────▶│                 │
│  ┌───────────────┐  │  HTTP   │  查询 API       │
│  │ 状态 API      │  │  GET    │  ↓             │
│  │ :8080/status  │  │         │  解析结果       │
│  │               │  │         │  ↓             │
│  │ 返回:         │  │────────▶│  飞书通知      │
│  │ - 进程状态    │  │         │  (异常/备份)   │
│  │ - 玩家数量    │  │         │                 │
│  │ - 内存使用    │  │         │                 │
│  │ - 最新备份    │  │         │                 │
│  └───────────────┘  │         │                 │
│                     │         │                 │
│  ┌───────────────┐  │         │                 │
│  │ 备份上传 OSS  │  │         │                 │
│  └───────────────┘  │         │                 │
└─────────────────────┘         └─────────────────┘
```

## 功能

### Server (Windows)

- HTTP API 查询服务器状态
- 玩家数量检测（解析日志）
- 自动备份（玩家离线时触发）
- 备份文件上传到阿里云 OSS

### Client (Mac)

- 定时查询 Windows API
- 异常状态飞书通知
- 从 OSS 下载备份文件

## 目录结构

```
server/           # Windows 端 API
  ├── api.py        # HTTP API 服务
  ├── monitor.py    # 状态检测逻辑
  ├── backup.py     # 备份 + OSS 上传
  └── config.yaml   # 配置文件

client/           # Mac 端监控脚本
  ├── monitor.py    # 定时查询 + 通知
  └── config.yaml   # 配置文件
```
